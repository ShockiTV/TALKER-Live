# bridge-remote-config

## Purpose

**DEPRECATED**: All requirements in this spec have been removed. The bridge no longer exists. URL configuration is now handled directly by Lua (`talker_ws_integration.script`) and the service (`talker_service` via pydantic-settings).

## Note

The bridge-remote-config spec is retained in the specs index for historical reference only. All bridge functionality has been migrated to direct Lua ↔ `talker_service` connections per the `direct-lua-service-connection` spec.
